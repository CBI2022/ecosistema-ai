Ayuda de búsqueda avanzada

¿Sabes que ahora también puedes buscar referencias de tus propiedades desde el buscador general?

Para hacerlo solo tienes que empezar tu búsqueda con el caracter #seguido de la referencia de la propiedad

Ejemplo referencia exacta:Para buscar la referencia exacta soo123tan solo escribe #soo123y la propiedad aparecera entre los resultados de búsqueda.

OTRAS FORMAS DE BUSCAR

De esta manera puedes encontrarreferencias exactas, pero también puedes usar el caracter *en tus búsquedas a modo de comodín si no recuerdas cual es la referencia exacta (siempre empezando con # para buscar entre las propiedades)

Ejemplo referencias que empiezan por:Para buscar referencias que empiecen por soopuedes escribir #soo*

Ejemplo referencias que terminan por:Para buscar referencias que terminan en 123puedes escribir #*123

Ejemplo referencias que contienen un texto:También puede ser que no conozcas como empieza ni como termina una referencia pero si una parte de ésta. En este caso solo debes utilizar el comodín antes y después del texto conocido, por ejemplo #*oo12*
